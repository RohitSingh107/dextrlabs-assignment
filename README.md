# Steps to run

## Requirements

- docker
- docker-compose

## Clone the repo

```
git clone https://github.com/RohitSingh107/dextrlabs-assignment
```

## Build and run the dockerized application

```
docker-compose up --build
```

# Avalable Enpoints

- POST /register
- POST /login
- POST /posts
- GET /posts
- GET /posts/:id
- PUT /posts/:id
- DELETE /posts/:id
- GET /posts/:id/comments
- POST /posts/:id/comments
